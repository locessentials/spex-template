# General Information

| **Parameter key** | **Parameter value** |
| ----- | ----- |
| Requestor/Requestor group | Fill in with the name of the requestor/requestor group |
| Website[^1] | ----- |
| Verticals & subverticals[^2] | ----- |
| Clientele | Broadly, who are the end users of the requester’s products and services? |
| Products/Services | What are the products and services that the organization offers to its clientele? |
| Nature of the product line | • Size: [in sales, number of products, etc.]<br>• General description of the services requested<br>• Scope of work: General volume [in words, pages, etc.] and approximate duration of time of the work [quarter, year, continous] |

## Notes

[^1]: A website URL that begins with https is secure, while a website URL that begins with http is not secure. See http vs. https via Snopes.com.
[^2]: Vertical is a synonym for subject field, subject matter, domain, discipline, etc.

## Using these Materials with GAI Tools

These specifications are written in [**Markdown format**](https://docs.github.com/en/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax) for easy integration with AI learning tools. Content written in markdown format can be better "understood" by generative AI tools, as the hierarchy and emphasis of content is clearly communicated through symbols. You can learn more about the philosophy behind the Markdown design here: [How to Use these Specifications](overview/how-to-use.md).
